# Phase 1 功能规格

## 界面布局

- **默认视图**：月历（Month Grid），应用启动即显示
- **顶部左侧**：当前年份与月份，格式 `YYYY/MM`（如 `2026/05`）
- **月历表格**：7 列网格，列顺序为 **周一至周日**（第一列为星期一）
- **月历行数**：按实际所需动态计算（`ceil((当月天数 + 月初偏移) / 7) × 7` 格），完整覆盖当月所有日期及前后月填充日期，不出现多余空白行

## 日期显示规则

| 规则 | 样式 |
|------|------|
| **当前月份日期** | 默认深色文字，正常显示 |
| **非当前月份日期** | 较暗灰色（`#bbb`），与本月日期明确区分 |
| **周末（周六、周日）** | 蓝色文字 |
| **节假日** | 蓝色文字（与周末统一），右上角绿色"休"标识 |
| **调休补班日** | 右上角红色"补"标识 |

每个日期格内显示两项内容：
- 上方：公历日期数字
- 下方：农历或节日信息，按以下优先级显示：
  - 若当日为节假日 → 显示节日名称（如"国庆节"），不显示农历
  - 若当日为农历初一 → 显示农历月份名称（如"三月"）
  - 其余日期 → 显示农历日称（如"初二"、"十五"、"廿三"）

## 两级高亮系统

当日（Today）与选中日期（Selected）使用**独立可叠加**的高亮样式：

| 状态 | 样式 | 触发条件 |
|------|------|----------|
| **今日基础高亮** | 蓝色边框 + 浅蓝底色 | 始终应用于当日日期，永不消失 |
| **选中高亮** | 深蓝底色 + 白色文字 + 外阴影 | 应用于当前选中日期 |

- 页面加载时：今日日期默认处于选中状态，同时叠加两种高亮（蓝色边框 + 深蓝底色）
- 点击其他日期时：选中高亮切换至该日期，今日的基础高亮始终保持
- 通过两种不同程度的视觉强调，清晰区分"今日"与"当前选中日期"

## 选中日期与天数差

当用户选中某一日期后，在顶部年份与月份区域右侧同步显示该日期距离今天的天数差：

- 格式示例：`2天后` / `2天前`
- 若今日日期被选中，则不显示天数差

在顶部区域最右侧提供**"回到今日"按钮**，点击后自动跳转至当前月份并选中今日。选中今日时此按钮隐藏。

## 壁纸设置

顶部右侧提供"壁纸设置"按钮（🖼），点击后弹出子菜单，提供以下功能：

### 导入图片

- 点击"📂 导入图片" → 打开系统文件选择器，筛选图片格式（jpg/png/bmp/webp）
- 选中图片后自动复制至应用 Resource 目录（`%APPDATA%/MySyncCalendar/Resource/wallpapers/`）
- 导入后立即设为当前壁纸

### 管理壁纸

- 点击"🖼 管理壁纸" → 展开 3 列缩略图网格，浏览所有已导入壁纸
- 点击缩略图 → 设为当前壁纸，当前选中壁纸有蓝色边框标识
- 点击缩略图右上角 ✕ 按钮 → 删除该壁纸
- 删除当前壁纸后自动切换至第一张可用壁纸

### 随机壁纸模式

- 勾选"随机壁纸模式" → 每次启动应用时从已有壁纸中随机选取一张
- 取消勾选 → 保持当前壁纸不变

## 年月快速跳转

点击顶部的年份与月份区域（如 `2026/05`），弹出年月选择器：

- **年份切换**：面板顶部提供 `<` / `>` 按钮，可切换目标年份
- **月份网格**：4×3 网格展示 1–12 月，当前浏览月份以高亮标识
- **选择月份**：点击任一月份按钮 → 关闭选择器，立即跳转至目标年月
- **关闭方式**：点击面板外部遮罩区域或按 `Esc` 键关闭

## 事项创建与编辑

顶部右侧提供"添加事项"按钮（`＋`），点击后弹出事项创建界面，支持两种事项类型：

### 年度事项

- **定义**：每年固定日期重复触发的事件（如生日、周年庆）
- **数据**：标题 + 月 + 日 + 可选提醒时间 + 可选备注（年选择器隐藏）
- **效果**：每年该日期的格子**左下角**显示橙色标记圆圈

### 单次事项

- **定义**：仅在指定日期发生的一次性事件
- **数据**：标题 + 年 + 月 + 日 + 可选提醒时间 + 可选备注 + 可选"纪念"属性
- **效果**：仅在设定日期的格子**右下角**显示绿色标记圆圈

日期选择统一使用三个独立的下拉选择器（年、月、日），根据事项类型自动显隐年选择器。

### 编辑事项

点击事件列表中事项旁的修改按钮（✏），弹出与添加事项相同的界面并自动填充所有字段。保存后更新原事项数据。

### 删除事项

点击事项旁的删除按钮（🗑），直接移除该事项。删除操作即时生效并持久化。

## 选中日期事件列表（月历视图）

当选中的日期存在已添加的事项时，日历网格下方区域自动列出该日期所有事项：

- 每条事项显示：类型标记 + 标题 + 备注（如有）+ 提醒时间（如有）+ 删除按钮
- 无事项时显示"无事项"占位文字
- 列表区域至少容纳两条完整事项而不需滚动

## 事件视图

底部视图切换栏（"月历" | "事件"）可切换至独立的事件列表视图：

### 筛选模式

- **今日之前（含今日）**：列出所有过往事件
- **今日之后（含今日）**：列出所有未来事件

### 置顶功能

- 每条事项左侧有图钉按钮（📌），点击后将事项置顶至列表最顶部
- 置顶事项应用浅黄色背景以示区分
- 再次点击取消置顶
- 支持多条同时置顶，按置顶操作先后顺序排列
- 置顶状态在两个筛选视图中保持一致

### 排序规则

- 置顶事项优先显示在上方（按置顶先后顺序）
- 其余事项按距今天数由近到远排列（天数绝对值升序）

### 相对天数计算

- **单次事项**：仅在其实际日期对应的视图中出现一次，显示距今实际天数
- **年度事项**：在"今日之前"视图显示距离最近一次过去日期的天数，在"今日之后"视图显示距离最近一次未来日期的天数

### 事件行内容

每行依次展示：置顶按钮（📌）、类型标记、标题、备注、提醒时间、距今天数、修改按钮（✏）、删除按钮（🗑）

## 交互操作

| 操作 | 行为 |
|------|------|
| 点击本月日期格 | 选中该日期，切换选中高亮，更新天数差显示 |
| 点击非本月日期格 | 自动跳转至对应月份，并选中该日期 |
| 点击年份月份区 | 弹出年月选择器，可快速跳转至任意年月 |
| `<` / `>` 按钮 | 切换至上/下一个月 |
| "＋" 添加事项按钮 | 弹出事项创建界面 |
| "🖼" 壁纸设置按钮 | 弹出壁纸设置子菜单（导入/管理/随机模式） |
| "回到今日"按钮 | 跳转至当前月份，自动选中今日，隐藏天数差及该按钮 |
| 底部视图切换栏 | 在"月历"和"事件"视图间切换 |
| 事件视图筛选标签 | 切换"今日之前/之后"筛选模式 |
| 置顶按钮（📌）| 置顶/取消置顶事项 |
| 修改按钮（✏）| 弹出编辑界面修改事项 |
| 删除按钮（🗑）| 删除事项 |
| 键盘 ← / → | 切换至上/下一个月 |
| 键盘 Esc | 关闭年月选择器或事项编辑界面 |
